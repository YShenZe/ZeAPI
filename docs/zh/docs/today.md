# 历史上的今天

## 1. 基本调用
访问以下URL格式获取当天历史事件：
```url
https://zeapi.ink/v1/today.php
```

## 2. 指定日期查询
添加查询参数指定月份和日期：
```url
https://zeapi.ink/v1/today.php?month=7&day=4
```

错误响应示例：
```json
{
    "status": "error",
    "message": "无效的日期参数: 月份 13 或日期 32 无效",
    "data": {
        "date": "",
        "events": []
    }
}
```

## 4. 响应字段说明

| 字段        | 类型   | 说明                          |
|-------------|--------|-----------------------------|
| status      | string | 请求状态（success/error）    |
| message     | string | 操作结果消息                 |
| data.date   | string | 查询的日期（月日）           |
| data.events | array  | 历史事件列表                 |

**事件对象字段：**

| 字段         | 类型   | 说明                          |
|--------------|--------|-----------------------------|
| year         | int    | 原始年份（负数为公元前）      |
| displayYear  | string | 格式化后的年份（含"公元前"） |
| type         | int    | 事件类型（1=事件，2=出生，3=逝世）|
| typeText     | string | 事件类型文本                 |
| data         | string | 事件描述内容                 |
