# On This Day in History

## 1. Basic Usage
Access the following URL to retrieve historical events for the current day:

```txt
https://zeapi.ink/v1/today.php
```

## 2. Specific Date Query
Add query parameters to specify a month and day:

```txt
https://zeapi.ink/v1/today.php?month=7&day=4
```

## 4. Response Fields Description

| Field        | Type   | Description                          |
|--------------|--------|------------------------------|
| status       | string | Request status (success/error) |
| message      | string | Operation result message      |
| data.date    | string | Queried date (month and day)  |
| data.events  | array  | List of historical events     |

**Event Object Fields:**

| Field        | Type   | Description                          |
|--------------|--------|------------------------------|
| year         | int    | Original year (negative for BC) |
| displayYear  | string | Formatted year (includes "BC" if applicable) |
| type         | int    | Event type (1=Event, 2=Birth, 3=Death) |
| typeText     | string | Event type text              |
| data         | string | Event description content    |
